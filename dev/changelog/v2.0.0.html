2.0.0¶

Enhancements¶

  • Added configuration option ALLOW_LONG_DIMSE_AET to allow bypassing the length check for elements with a VR of AE (#515)

  • Added the ability to pass a file path to a dataset to send_c_store() (#517)

  • Added STORE_SEND_CHUNKED_DATASET config option to allow the use of a chunked read when a file path is passed to send_c_store() in order to reduce the memory needed for sending large dataset (#517)

Changes¶

  • Removed support for Python 2.7 and 3.5

  • Minimum pydicom version is 2.0
